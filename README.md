#### `yarn dev`

Running app on http://localhost:3000

#### `yarn build`

#### `yarn export`

Generate static site from previous build

#### `yarn build:static`

Build app then generate static version

#### `yarn start`

Running app for production after build

#### `yarn lint`

Run ESLint for check your codebase
